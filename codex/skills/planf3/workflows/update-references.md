# Update References

1. Найди target Markdown plan.
2. Найди связанные docs/plans и тип связи: back reference или forward reference.
3. Обнови раздел Metadata/References без дубликатов.
4. Если обновляешь обе стороны связи — добавь reciprocal reference.
5. Не меняй implementation content.
6. Добавь запись в Amendments.
7. Сообщи список изменённых файлов и ссылок.
